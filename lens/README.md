# Lens

> Symbol-aware code map for Claude. Drop-in replacement for `graphify` in the `super-coder` skill, with an explicit "Ctrl+Click" primitive (`lens follow <symbol>`) so Claude can pull minimal slices instead of whole files.

**Status:** under development.

See the [`super-coder` skill](https://github.com/sudeep-dasgupta/claude-skill) for the loop Lens plugs into.
